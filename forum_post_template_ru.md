# Room Control Center - Центр мониторинга для REAPER

**Room Control Center (RCC)** - докуемый центр мониторинга, измерения уровня, воспроизведения референсов и коррекции наушников для REAPER.

## Возможности

- **Управление мониторами** - Переключение между комплектами мониторов (A/B/C), dim, mono
- **Измерение уровня** - Реалтайм измерение уровня с громкостью BS.1770/EBU R128
- **Спектроанализатор** - Визуализация частотного спектра с перцептивной ресемплингом
- **Отображение waveform** - Реалтайм отображение waveform с вертикальным зумом
- **Пространственная панель** - Мониторинг环绕/пространственного звука
- **Калибровка наушников** - Коррекция EQ для наушников со встроенными профилями
- **Референсный воспроизведение** - Воспроизведение референсного трека с отображением waveform
- **Докуемый интерфейс** - Докинг в любой docker REAPER

## Установка

### Через ReaPack (рекомендуется)

1. Установите [ReaPack](https://reapack.com/) и **ReaImGui**
2. В REAPER откройте `Extensions > ReaPack > Import repositories`
3. Добавьте: `https://raw.githubusercontent.com/uncleosmixing/ReaScripts/main/index.xml`
4. Синхронизируйте пакеты
5. Найдите **Room Control Center** и установите
6. Перезапустите REAPER
7. Запустите установленное действие из Action List

### Ручная установка

1. Скачайте ZIP-архив
2. Извлеките в папку Scripts REAPER
3. Добавьте скрипт в Action List REAPER

## Включенные сторонние компоненты

Room Control Center включает **Roof|control** от Ilya-audio (лицензия MIT) - система мониторинга наушников с эмуляцией колонок, кроссфидом и калибровкой наушников.
