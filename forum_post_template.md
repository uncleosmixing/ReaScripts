# Room Control Center - Dockable Monitoring Center for REAPER

**Room Control Center (RCC)** is a dockable monitoring, metering, reference playback and headphone correction control center for REAPER.

## Features

- **Monitor Control** - Switch between monitor sets (A/B/C), dim, mono
- **Level Metering** - Real-time level meters with BS.1770/EBU R128 loudness
- **Spectrum Analyzer** - Frequency spectrum visualization with perceptual resampling
- **Waveform Display** - Real-time waveform view with vertical zoom
- **Spatial Panel** - Surround/spatial audio monitoring
- **Headphone Calibration** - EQ correction for headphones with built-in profiles
- **Reference Player** - Reference track playback with waveform display
- **Dockable Interface** - Dock to any REAPER docker

## Installation

### Through ReaPack (recommended)

1. Install [ReaPack](https://reapack.com/) and **ReaImGui**
2. In REAPER open `Extensions > ReaPack > Import repositories`
3. Add: `https://raw.githubusercontent.com/uncleosmixing/ReaScripts/main/index.xml`
4. Synchronize packages
5. Search for **Room Control Center** and install
6. Restart REAPER
7. Run the installed action from the Action List

### Manual installation

1. Download the ZIP archive
2. Extract to your REAPER Scripts folder
3. Add the script to REAPER's Action List

## Dependencies

- REAPER 7
- ReaPack
- ReaImGui

## Included Third-Party
