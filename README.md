# Room Control Center

Dockable monitoring, metering, reference playback and headphone correction control center for REAPER.

## Installation through ReaPack

1. Install [ReaPack](https://reapack.com/) and **ReaImGui**.
2. In REAPER open `Extensions > ReaPack > Import repositories`.
3. Add:

   `https://raw.githubusercontent.com/uncleosmixing/ReaScripts/main/index.xml`

4. Synchronize packages.
5. Search for **Room Control Center**.
6. Install **Room Control Center**.
7. Restart REAPER.
8. Run the installed action from the Action List.

## Features

- **Monitor Control** - switch between monitor sets (A/B/C), dim, mono
- **Level Metering** - real-time level meters with BS.1770/EBU R128 loudness
- **Spectrum Analyzer** - frequency spectrum visualization
- **Waveform Display** - real-time waveform view
- **Spatial Panel** - surround/spatial audio monitoring
- **Headphone Calibration** - EQ correction for headphones
- **Reference Player** - reference track playback with waveform
- **Dockable Interface** - dock to any REAPER docker

## Dependencies

- REAPER 7
- ReaPack
- ReaImGui

## Included Third-Party

Room Control Center bundles **Roof|control** by Ilya-audio (MIT License) - a headphone monitoring system with speaker emulation, crossfeed and headphone calibration.
