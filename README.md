# Room Control Center

Dockable monitoring, metering, reference playback and headphone correction control center for REAPER.

## Installation

1. Install [ReaPack](https://reapack.com/) and **ReaImGui**.
2. In REAPER: `Extensions > ReaPack > Import repositories`.
3. Add: `https://raw.githubusercontent.com/uncleosmixing/ReaScripts/main/index.xml`
4. Synchronize packages.
5. Install **Room Control Center**.
6. Restart REAPER.

## Features

- Monitor Control (A/B/C switching, dim, mono)
- Level Metering (BS.1770/EBU R128)
- Spectrum Analyzer
- Waveform Display
- Spatial Panel
- Headphone Calibration
- Reference Player
- Dockable Interface

## Dependencies

- REAPER 7
- ReaPack
- ReaImGui

## Bundled Third-Party
